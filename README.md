## Installation & Getting started

1. Clone this repository
    ```
    git clone https://github.com/Sayli555/wanderonBe
    ```

2. Install dependencies:

    ```
    npm install
    ```
3. Run the development server:
    ```
    npm run server
    ```

This will start your server.
